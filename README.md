# 🧵 Thread Pool Simulator

A visual, interactive simulator that brings concurrent programming to life.
Submit tasks, watch worker threads execute them in priority order, scale 
the pool up and down, and monitor real-time metrics — all running directly 
in your browser.

## 🔍 What It Shows
- How thread pools reuse threads instead of creating new ones per task
- Priority queue — Critical tasks run before Low tasks automatically
- Dynamic scaling — add or remove threads live
- What happens when the queue overflows (rejection policies)
- Real-time metrics: active threads, queue depth, completed, rejected

## 🚀 How to Run
Just open `index.html` in any browser. No installation needed.

Or visit the live demo:
👉 https://YOUR_USERNAME.github.io/thread-pool-simulator

## 🛠️ Built With
- HTML5
- CSS3
- Vanilla JavaScript

## 📚 Concepts Covered
- Thread lifecycle (idle → busy → idle)
- Priority-based task scheduling
- Thread pool auto-scaling
- Task rejection policies
- Real-time performance monitoring
